# el-pollo-en-el-espacio
One try of my homepage

## Requirements
+ chicken 5
+ chicken eggs:
  + spiffy
  + spock
  + sxml-serializer
+ GNU make
+ ssl certificate

## Installation

`$ mkdir certs`

`$ cp <path to your public ssl cert> ./certs/cert.pem`

`$ cp <path to your ssl key> ./certs/key.pem`

`$ make all`

and

`$ ./a.out`

or

`$ make run`
